云原生 在公有云、私有云和混合云等新型动态环境中，构建和运行可弹性扩展的应用。云原生的代表技术包括容器、服务网格、微服务、不可变基础设施和声明式 API

Kubernetes

Kubernetes 是云原生操作系统，一个可移植、可扩展的开源平台，用于管理容器化的工作负载和服务，进行声明式配置和自动化。

DevOps

DevOps 是一种思想、一组最佳实践、一种文化。实现软件开发、测试、运维的一体化，减少部门之间的沟通成本，实现软件的快速高质量的发布。

Istio

Istio 是一个开源服务网格，Istio 与 Kubernetes 和传统工作负载一起使用，为复杂的部署带来了标准的通用流量管理、遥测和安全性。
